PubTal Manual Contents

Contents of the PubTal manual.

The PubTal manual is divided into several different sections.  If you are new to PubTal it is probably easiest to start with the Quick Start section.

  • Quick Start - A quick start guide to using PubTal.
  • Installation Guide - A guide to installing PubTal on Linux, MacOS X and Windows.
  • Template Basics - Describes the basics of the PubTal templating system.
  • Authoring Web Content
    • HTMLText - How to author web pages using the HTMLText content type.
    • OpenOffice - How to author web pages using OpenOffice.
    • Catalogues - How to create Photo Albums using PubTal's Catalogue support.
    • Raw & Binary - What are the Raw and Binary content types?
    • Weblog - Maintaining a Weblog using PubTal.
  • Configuration - Configuration options in PubTal.
  • Uploading - How to make PubTal upload your website.
  • Optional Plugins - A guide to the optional plugins.
  • Plugin API - How to write new plugins.
  • Commands Reference - Reference guide for the commands.
